Saker LUA API 文档

@作者: cinience
@日期: 2013-10-09

Saker 是系统服务级软件, 基于LUA扩展. Fock Me at github.com/cinience. 本文档介绍了 saker 的 LUA API.

saker.sysinfo(key, param)

本节介绍saker.sysinfo的不种参数的意义

system.boottime

说明

获取系统启动时间.

参数

返回值

出错则返回 nil,err, 其它值表示正常.成功则返回int

示例

  boottime,err = saker.sysinfo("system.boottime")
  print(boottime)
